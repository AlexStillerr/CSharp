# C# Projects
A collection of small C# projects created for learning and practicing C#, without using Unity.

## Console Adventure
A text-based console adventure featuring small mini-games.
The game text is written in German.

## Match Four
A WPF implementation of the classic Match Four game.
Two human players can play against each other.

This project is also used to train an AI via a Python server (separate project).

## Quiz
A learning quiz application built with Windows Forms.
The questions are generated by ChatGPT and are written in German.
